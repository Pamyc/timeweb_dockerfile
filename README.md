# Next.js + Dockerfile for Timeweb Cloud (Apps)

Minimal Next.js (App Router) project with a Dockerfile optimized for Timeweb Cloud "Apps > Dockerfile".

## Run locally
```bash
npm install
npm run dev
# open http://localhost:3000
```

## Build & run with Docker
```bash
docker build -t timeweb-next .
docker run -p 3000:3000 -e PORT=3000 timeweb-next
```
